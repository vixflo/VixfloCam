# Analiză - Redesign UI (stil modern)

## Obiectiv

- Stilizare consistentă și modernă:
  - meniuri, butoane, controale, culori, layout
  - logo și identitate vizuală

## Direcții recomandate

- QSS theme (dark modern) + variabile de culoare.
- Separare „Live” vs „Events” cu toolbar clar.
- Componentizare: widget-uri dedicate (PTZ panel, Events panel, Settings dialog).
